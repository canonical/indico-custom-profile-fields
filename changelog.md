# Changelog

### 2025-12-18

- Separate the field "Legal Name" into "Legal First Name" and "Legal Last Name".
- Add the field "Country" to the profile. It matches to the "Country" field in registration forms.
- Employee ID and CAP details are now admin-only. They can be changed only by admins, and are not visible in registration forms.

### 2025-10-31

- docs: Contributing guide and changelog is added. Vale workflows are added.

### 2025-10-03

- Version 1.0.0 is released as the first working version of Custom Profile Fields plugin.
